h1 até h6

h1-h6: Os elementos HTML de cabeçalho da seção

Os elementos HTML h1-h6 representam seis níveis de título de seção.

h1 é o nível de seção mais alto e h6 é o mais baixo.

p

O elemento HTML p representa um parágrafo. Em mídias visuais,

parágrafos são representados como blocos indentados de texto com a primeira letra avançada

e separados por linhas em branco.

mark

O Elemento HMTL mark representa um trecho de destaque em um texto, por exemplo,

uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

small

O elemento small agora é empregado para marcar avisos obrigatórios,

restrições legais ou restrições de uso de conteúdo como copyrights,

seja, não é apenas para tornar o tamanho do texto menor.

i

representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos,

expressões de outros idiomas ou pensamentos de personagens fictícios.

u

representa uma lista de itens sem ordem rígida, isto é,

uma coleção de itens que não trazem uma ordenação numérica e as suas posições,

nessa lista, são irrelevantes.

strong

dá ao texto uma forte importância, e é tipicamente mostrado em negrito

ol

representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista

são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras,

algarismos romanos, ou simples símbolos.

ul

representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não

trazem uma ordenação numérica e suas posições, nessa lista, são irrelevantes.

il

representa uma lista de itens ordenados.

a

com o atributo href cria-se um hiperligação nas páginas web,

arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

hr

representa uma quebra temática entre elementos de nível de parágrafo (por exemplo ,

uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML,

representava uma linha horizontal

sub

destaca palavras com conceitos visíveis por meio da interação. Usamos a tag HTML:

u. Tachado: sugere a eliminação de conceitos nos processos de edição ou representa uma ironia.

sup

Um trecho do texto pode ser formatado como subscrito ou sobrescrito usando-se as tags sub ou sup.

Essas tags são muito úteis em fórmulas químicas e em algumas equações matemáticas. É só copiar e colar o código em seu conteúdo!

blockquote

indica que o texto incluído é uma longa citação. Normalmente,

este é processado visualmente pelo recuo (ver Notas sobre como mudá-lo).

front

É uma tag vazia, que contém apenas atributos e faz a relação do documento HTML com recursos externos.

É comumente usado para vincular uma folha de estilo externa, também é usada para definir o favicon da página

(ícone da aba do navegador), como outros recursos.

del

representa uma parte do texto que foi excluída de um documento.

Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

abbr

indica uma abreviatura ou sigla, como “WWW” ou “NATO”.

Marcando-se abreviaturas que você pode dar informações úteis para os navegadores,

corretores ortográficos, sistemas de tradução e indexadores.
